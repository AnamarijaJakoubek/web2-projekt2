Cross-site scripting (XSS)

Što je XSS napad?

XSS (Cross-Site Scripting) napad omogućava napadaču umetanje zlonamjernog JavaScript koda u web-stranicu koju će vidjeti drugi korisnici. Ovaj napad može rezultirati krađom podataka poput kolačića, manipulacijom sadržajem stranice ili preusmjeravanjem korisnika na maliciozne stranice.

Upute:

  1. Uključivanje/isključivanje XSS ranjivosti: Označavanjem checkboxa korisnik može uključiti ili isključiti XSS ranjivost.
  2. Sanitizacija unosa: Kada je ranjivost isključena, sanitizacija se provodi zamjenom znakova &, ', ", <, >, / i ` s odgovarajućim HTML entitetima kako bi se spriječio napad.
  3. Omogućena ranjivost: Kada je ranjivost uključena, sanitizacija se ne provodi, što omogućava napadaču unos malicioznog koda.

Primjeri korisničkih unosa:

  1. <IMG SRC=# onmouseover="alert('XSS')">
  2. <a href="javascript:alert('XSS')">Klikni me</a>
  3. <a onmouseover="alert(document.cookie)">xxs link</a>
  4. <textarea oninput="alert('XSS')">Type here</textarea>
  5. <div style="width:100px; height:100px;" onclick="alert('XSS')">Klikni me</div>
  6. <iframe src="javascript:alert('XSS')"></iframe>
  7. <img src=x onerror="alert('XSS')">
